# Locadora de Veículos – Desafio com Microserviços

Este repositório contém a implementação de uma aplicação de **locadora de veículos**, desenvolvida como desafio durante o treinamento dos estagiários da Solutis na trilha de desenvolvimento de software.
O foco do projeto está na aplicação de conceitos de **arquitetura de microserviços**, utilizando **Java com Spring**.

A solução contempla o fluxo principal de uma locadora:  
- Cadastro de clientes;
- Consulta e seleção de veículos disponíveis;
- Reserva e efetivação de aluguel.

O projeto foi pensado para simular um cenário real de desenvolvimento em equipe, promovendo o uso de boas práticas como versionamento de código, documentação técnica, definição de decisões arquiteturais e separação de responsabilidades entre serviços.

Toda a documentação do projeto está versionada neste repositório, servindo como apoio técnico e histórico de evolução da solução.

O projeto serve como ambiente de aprendizado prático para trabalho colaborativo, versionamento, documentação e aplicação de conceitos comuns em sistemas distribuídos.
