# US-02 – Escolha de Veículo para Aluguel

## Descrição
**Como** um cliente cadastrado na locadora de automóveis  
**Quero** poder escolher um veículo disponível para alugar  
**Para** reservar um veículo que atenda às minhas necessidades de locomoção

## Regras de Negócio
- Apenas veículos disponíveis podem ser selecionados para aluguel.
- O período de aluguel deve possuir data de início e término válidas.
- O valor do aluguel deve ser calculado com base no período selecionado.

## Critérios de Aceite
- [ ] Deve existir uma seção de seleção de veículos na página inicial.
- [ ] Veículos disponíveis devem ser exibidos com informações básicas (fabricante, modelo, categoria, acessórios e preço por dia).
- [ ] Cada veículo deve possuir uma imagem representativa.
- [ ] O cliente deve conseguir aplicar filtros por categoria e acessórios.
- [ ] Ao selecionar um veículo, o cliente deve visualizar seus detalhes.
- [ ] O cliente deve escolher o período de aluguel.
- [ ] O veículo selecionado deve poder ser adicionado ao carrinho de aluguel.
- [ ] O carrinho deve exibir veículo, datas e custo total estimado.
- [ ] O cliente deve conseguir revisar e confirmar a reserva.
- [ ] Após confirmação, deve haver uma confirmação visual da reserva.

## Observações
- O carrinho de aluguel representa uma etapa intermediária antes da efetivação do aluguel.
